Native Deeplink Management for Android

🚀 Features

📂

Save & Organize Deeplinks

Store, categorize, and manage your deeplinks for instant access.

⚡

Launch Deeplinks

Test and launch deeplinks directly from the app to speed up your workflow.

🔍

Search & Sort

Find the right deeplink instantly with powerful search and sorting.

📊

Open Counter

Track usage frequency for each deeplink to gain testing insights.

🏠

Home Screen Shortcuts

Add deeplinks to your Android home screen for one-tap access.

🔒

Privacy First

Your data stays on your device. No tracking, no ads, just productivity.

🏗️ Architecture

Deepr is built with modern Android technologies for performance, scalability, and developer happiness.

🎨 Jetpack Compose Declarative & reactive UI toolkit for beautiful interfaces.
🧭 Compose Navigation Type-safe navigation for seamless user flows.
🗂️ Android ViewModel Lifecycle-aware state management for robust apps.
🗄️ SQLDelight Type-safe, multiplatform SQL database for reliability.
🔗 Koin DI Simple dependency injection for modular code.
⏱️ Kotlin Coroutines Efficient background operations for smooth performance.
