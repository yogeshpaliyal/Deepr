üõ°Ô∏è Privacy Policy

Deepr is built with a privacy-first philosophy. As an open-source tool designed for developers, we believe your data should belong to you and stay under your control.

1. Data Collection

Deepr does not collect, store, or transmit any personal information to external servers. All deeplinks, categories, and usage statistics are stored locally on your device's private storage using SQLDelight.

2. Analytics & Tracking

The application contains no trackers, no advertising SDKs, and no third-party analytics. We do not monitor how you use the app or what links you store.

3. Permissions

Deepr requests minimal permissions required to function (such as creating Home Screen shortcuts). It does not request access to your contacts, location, or files unless explicitly triggered by a user action for a specific feature.

4. Open Source Transparency

Because Deepr is open-source, you can verify our privacy claims by reviewing the source code on our GitHub repository at any time.
